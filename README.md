Generative Search Engine

About This Project

I built this project to explore how modern AI systems and search engines can work together in a single application.

Most search engines provide a list of links, while AI assistants generate answers. I wanted to create a system that can intelligently decide which experience is more useful based on the user's query.

The project detects user intent and either:

- Generates an AI response for knowledge-based questions.
- Returns relevant web search results for search-oriented queries.

This project helped me understand full-stack development, API integration, deployment, frontend-backend communication, and practical AI application development.

---

Why I Built This

While using search engines and AI chatbots, I noticed that both have strengths and weaknesses.

- Search engines provide sources but require users to read through multiple websites.
- AI assistants provide direct answers but sometimes lack source discovery.

I wanted to experiment with combining both approaches into a single system and learn how real-world AI products are built.
